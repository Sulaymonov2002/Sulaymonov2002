4 -> HTML-BASICS
    🔷 Questions
         ✅ Html nima va uning yoyilmasi ? Javob: HyperText Markup Language. Gipermatinni hisoblash tili hisoblanadi.
         ✅ Web saytlar nechta qisimdan tashkil topadi ?  Javob: Websaytlar Fronted va Backenddan tashkil topgan.
         ✅ Web saytlar qanday prinsip asosida ishlaydi ? Javob: Kilent so'rov jo'natadi Web sayt javob beradi.
         ✅ HTML kodlarimiz ichidagi ushbu tag [<title>Mening Sahifam</title>] vazifasi nima ? Javob: Sahifaning majburiy elementi <title> </title> teglari hisoblanadi.
         ✅ Xozirda biz HTML ning nechanchi versiasidan foydalanamiz Javob: HTML5
         ✅ HTML kodlarimiz ichidagi ushbu tag [<meta charset="UTF-8" />] vazifasi nima ? Javob: UNICODE asosida ishlaydigan kodlash jadvali.
         ✅ Absolute path va relative path larga bittadan misol yozing. absolute:  <img src="https://upload.wikimedia.org/wikipedia/commons/thumb/6/61/HTML5_logo_and_wordmark.svg/1200px-HTML5_logo_and_wordmark.svg.png" height="200px" width="200px">
         relative:  <img src="images/river.jpg" height="200px" width="200px">
         ✅ Biz loyihamizdagi rasmni yuklashimiz uchun bizga Absolute path kerakmi yoki Relative path ? Javob: Absolute path kerak bo'ladi.
         ✅ Block element va Inline Element farqlarini tushuntirib bering ? Javob: Block elementlar har doim yangi qatordan boshlanadi.
Block elementlar o'zi turgan qatorni to'liqligicha egallaydi (iloji boricha chapga va o'ngga cho'ziladi).
Block elementlar tepa va pastdan kenglikka (margin) ega bo'ladi.
Inline elementlar yangi qatordan boshlanmaydi.
Inline elementlar faqat o'ziga keraklicha bo'lgan joyni oladi.
Inline elementlar tepa va pastdan kenglikka (margin) ega bo'lmaydi.
         ✅ HTML da block va inline elementlar uchun bir nechta misollar keltiring. Javob: <h1>Block element</h1>; <b>Qalin</b> va <i>yotiq</i> matn.
